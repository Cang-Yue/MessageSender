# MessageSender
A Duplicate message sending tool

## How to use
1. run this program
2. input data
3. move mouse focus to a Input box
4. Wait

## How to build
Install some package
```
pip install random
pip install pyperclip
pip install time
pip install pynput
pip install tqdm
```
run `build.sh` or `build.cmd`

You can find the file in `./dist/`
